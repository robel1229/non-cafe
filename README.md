# 🍲 Non-Cafe System (Ethiopian Edition)

A high-performance, mobile-first web application built specifically for university students in Ethiopia to manage "Non-Cafe" meal expenses easily. 



## ✨ Key Features
* 📅 **Ethiopian Calendar (E.C):** Automatically tracks entries using the local calendar system.
* 🍳 **Meal Categories:** Support for Breakfast, Lunch, and Dinner.
* 👥 **Dynamic Group Management:** Add any number of group members on the fly.
* 💰 **Smart Split:** Automatically calculates individual shares with detailed brackets (e.g., `Aman(75.50)`).
* 📊 **Monthly Summary:** A dedicated dashboard showing the total amount each student needs to pay.
* 📥 **Excel Export:** Download your entire history as a professional `.xlsx` file for offline records.
* 📱 **Offline Persistence:** Uses `LocalStorage` to keep your data saved on your phone even without an internet connection.

## 📖 How to Use
1.  **Add Members:** Enter the names of your group members.
2.  **Enter Details:** Put in the location (e.g., Wude) and the total price of the meal.
3.  **Select Who Ate:** Check the boxes for the students who shared the meal.
4.  **Save:** Click "Save Daily Record" to add it to your history.
5.  **Settle Up:** Check the "Monthly Summary" at the bottom to see who owes what.
